IDEA常用配置（持续更新）

发表于2023-04-11|更新于2023-04-11|IDEA
|阅读量:

常用配置

  1. 修改字体：Appearance & Behavior --> Appearance --> Use custom font
  2. 修改代码字体：Edit --> Font
  3. 修改快捷键：Keymap
  4. 禁止更新：Appearance & Behavior --> System Setting --> Updates --> 取消 Check IDE updates for、Check for plugin updates(可选)、Show What’s New in the editor after an IDE update
  5. 忽略大小写提示：Editor --> General --> Code Completion --> 取消 Match case
  6. 选中代码禁止拖拽：Editor --> General --> 取消 Move code fragments with drag-and-drop
  7. 单行注释不从行首开始：Editor --> Code Style -> Java --> Code Generation --> Comment Code --> 取消 Line comment at first clumn、选中 Add a space at line comment start(可选)
  8. 文件编码：Editor --> File Encodings
  9. 隐藏文件：Editor --> File Types --> Ignored Files and Folders --> 添加对应的文件就行，比如.idea、target、out、*.iml
  10. 去除XML文件SQL黄色、绿色背景：Editor --> Inspections --> SQL --> 取消 No data sources configures、SQL dialect detection；Editor --> Color Scheme --> General --> Code --> Injected Ianguage fragment --> 取消Background
  11. 忽略单词拼写检查：Editor --> Inspections --> 取消 Proofreading
  12. 忽略注释里的错误警告：Editor --> Inspections --> Java --> Javadoc --> 取消Dangling Javadoc comment、Declaration has problems in Javadoc references、Javadoc declaration problems、Missing Javadoc
  13. 新文件加文件注释头：Editor --> File and Code Templates --> Includes --> File Header
  1. Maven、Gradle：Build, Execution, Deployment --> Build Tools
  2. Git：Version Control --> Git
  3. 去除任务栏IDEA小图标：Appearance & Behavior --> Notifications --> 取消 Enable system notifications

插件

  1. GitToolBox：git工具箱
  2. Rainbow Brackets：彩虹括号
  3. MyBatisX
  4. 力扣刷题插件：Leeocodeeditor
